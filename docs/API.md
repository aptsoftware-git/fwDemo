# FRS Data Management System - API Documentation

## Base URL

```
http://localhost:8000/api
```

## Authentication

No authentication required (as per system requirements).

## Endpoints

### 1. Upload Directory

**Endpoint**: `POST /api/upload`

**Description**: Upload and process a directory containing FRS data from multiple units.

**Request Body**:
```json
{
  "directory_path": "c:\\data\\FRS_Sample\\Nov",
  "tag": "Fmn November 2025",  // optional
  "description": "November readiness data"  // optional
}
```

**Response** (Success):
```json
{
  "success": true,
  "message": "Successfully processed 2 unit(s)",
  "dataset": {
    "id": 1,
    "tag": "Fmn November 2025",
    "upload_date": "2026-03-04T10:30:00Z",
    "month_label": "November",
    "description": "November readiness data",
    "unit_count": 2
  },
  "units_processed": 2,
  "errors": []
}
```

**Response** (With Warnings):
```json
{
  "success": true,
  "message": "Successfully processed 2 unit(s) with 1 warning(s)",
  "dataset": { ... },
  "units_processed": 2,
  "errors": [
    "No Excel files found in: Fmn C Nov"
  ]
}
```

**Response** (Error):
```json
{
  "success": false,
  "message": "Directory not found: invalid_path",
  "units_processed": 0,
  "errors": ["Directory does not exist"]
}
```

### 3. Get Data

**Endpoint**: `GET /api/data/{tag}`

**Description**: Retrieve data for a specific dataset with optional filtering.

**Path Parameters**:
- `tag` (required): Dataset tag (URL encoded if contains spaces)

**Query Parameters**:
- `unit_filter` (optional, default: "All"): Filter by unit name ("All" or specific unit like "Fmn A")
- `sheet_type` (optional): Filter by specific sheet type (e.g., "APPX_A_BVEH")

**Example Request**:
```
GET /api/data/Fmn%20November%202025?unit_filter=All
GET /api/data/Fmn%20November%202025?unit_filter=Fmn%20A&sheet_type=APPX_A_BVEH
```

**Response**:
```json
{
  "tag": "Fmn November 2025",
  "unit_filter": "All",
  "sheets": {
    "APPX_A_BVEH": [
      {
        "Column_1": "Value1",
        "Column_2": 42,
        "Column_3": "2025-11-01"
      },
      ...
    ],
    "APPX_A_CVEH": [...],
    "ARMT": [...],
    "SA": [...],
    "INST": [...],
    "CBRN": [...]
  }
}
```

---

### 4. Get Units

**Endpoint**: `GET /api/data/{tag}/units`

**Description**: Get list of units in a dataset.

**Path Parameters**:
- `tag` (required): Dataset tag

**Example Request**:
```
GET /api/data/Fmn%20November%202025/units
```

**Response**:
```json
["Fmn A", "Fmn B"]
```

---

### 5. Compare Datasets

**Endpoint**: `POST /api/compare`

**Description**: Compare two datasets and generate LLM-powered analysis.

**Request Body**:
```json
{
  "tag1": "Fmn November 2025",
  "tag2": "Fmn December 2025",
  "sheet_types": null  // optional: array of specific sheet types to compare
}
```

**Response**:
```json
{
  "tag1": "Fmn November 2025",
  "tag2": "Fmn December 2025",
  "comparison_text": "## Comparison Analysis\n\nBased on the data comparison between November and December 2025:\n\n### APPX_A_BVEH\n- Overall readiness increased by 12%\n- 5 new vehicles added\n- 2 vehicles removed from service\n...",
  "generated_at": "2026-03-04T11:30:00Z"
}
```

**Notes**:
- This endpoint may take 30-60 seconds to complete as it waits for LLM processing
- The `comparison_text` contains detailed line-by-line analysis generated by the LLM

---

### 6. Health Check

**Endpoint**: `GET /api/health`

**Description**: Check if the API is running.

**Response**:
```json
{
  "message": "Service is healthy",
  "success": true
}
```

## Error Responses

All endpoints may return error responses in the following format:

**404 Not Found**:
```json
{
  "detail": "Dataset not found: Invalid Tag"
}
```

**500 Internal Server Error**:
```json
{
  "detail": "Database connection failed"
}
```

**422 Unprocessable Entity** (Validation Error):
```json
{
  "detail": [
    {
      "loc": ["body", "directory_path"],
      "msg": "field required",
      "type": "value_error.missing"
    }
  ]
}
```

---

## Sheet Types

The system recognizes the following sheet types:

| Sheet Type Key | Description |
|---------------|-------------|
| `APPX_A_BVEH` | Appendix A (B Vehicles) |
| `APPX_A_CVEH` | Appendix A (C Vehicles) |
| `ARMT` | Armament |
| `SA` | Small Arms |
| `INST` | Instruments |
| `CBRN` | CBRN Equipment |
